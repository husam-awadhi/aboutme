# aboutme
simple portfolio application using Remix framework and Mongodb.

## Installation
run: 
```sh
docker-compose build
```
then 
```sh 
docker-compose up
```

## Access

`localhost:3000`

## Remix

- [Remix Docs](https://remix.run/docs)

### Development

From your terminal:

```sh
npm run dev
```

### Deployment

First, build your app for production:

```sh
npm run build
```

Then run the app in production mode:

```sh
npm start
```
