# aboutme
simple portfolio application using Remix framework and Redis as Database.

## Prerequisite
* Docker
## Installation
build: 
```sh 
docker-compose build
```
run: 
```sh 
docker-compose up
```

## Access

`http://localhost:3000`

## Dependencies
- [Remix Documentation](https://remix.run/docs)
- [Redis Documentation](https://redis.io/documentation)
